Graduate Course Description

MCS5023 Java Programming - 3 credits
Prerequisite:   C++ programming and Data Structures

An introduction to Java programming covering the basic elements of Java 
as well as principles of object-oriented programming.  Emphasis on 
developing Applets for the Internet and GUI-based applications.

MCS5103 Introduction to Software Engineering - 3 credits
Prerequisite:  permission of department chairman

Selection of programming language; debugging techniques and tools; program maintenance; 
software economics; team programming and its application to projects; software life cycle.

MCS5203 Computer Graphics 1 - 3 credits
Prerequisite: C++ and Data Structures

Introduce the concepts underlying two-dimensional and three-dimensional computer graphics.  
Topics include basic architecture, interactive techniques, coordinate systems, 
transformations, and illumination.

MCS5303 Introduction to Database Systems - 3 credits
Prerequisite:  permission of department chairman

Design and implementation of relational, hierarchical and network data base systems.   
Query/update data languages, conceptual data model, physical storage methods, data 
base system architecture and normal forms.  Data base security and integrity.  
Relational data base systems  are emphasized.  A project involving an on-line data 
base systems is normally assigned.

MCS5503 Introduction to Artificial Intelligence - 3 credits
Prerequisite:  permission of department chairman

Introduction to languages LISP AND PROLOG and techniques of artificial intelligence; 
development of programs in LISP and PROLOG to illustrate problem-solving mechanisms; 
problem definition using state-space techniques; problem solving heuristics; 
inference in  monotonic and non-monotonic logic; knowledge representation technique; 
discussions of application in various areas.

MCS5703 Introduction to Distributed Computing - 3 credits
Prerequisite:  permission of department chairman

Introduction to data communications, network models, topologies and structures.  
This will include the OSI model, transport mediums (routers, bridges, gateways), 
and an overview of communication protocols, particularly TCP/IP.

MCS6123 Advanced Software Engineering Techniques - 3 credits
Prerequisite: MCS5103

Architecture of software environments.  Syntax directed editors. Tools for programming 
in-the-large.Tools to support the assessment of partial designs. Expert systems for 
software development.

MCS6143 Current Developments in Software Engineering - 3 credits
Prerequisite: MCS6123

Seminar in Software Engineering,  discussion of current papers in the field.  

MCS6323 Distributive Database Systems - 3 credits
Prerequisite: MCS5303

Relational operators; dependency theory; project-join mappings; representation theory; 
query optimization; null values; distributed databases; concurrence control.  
Distributed data bases; networking; office information systems; distributed vision applications.

MCS6343 Current Developments in Database Systems - 3 credits
Prerequisite: MCS6323

Seminar in Database Management Systems, discussion of current papers in the field.

MCS6523 Expert Systems - 3 credits
Prerequisite: MCS5503

Advanced topics in development and implementation of intelligent systems; 
knowledge acquisition and representation for expert systems; design of adaptable 
expert systems that learn.


MCS6543 Current Developments in Artificial Intelligence - 3 credits
Prerequisite: MCS6523

Seminar in Artificial Intelligence Systems, discussion of current papers in the field.  

MCS6723 Advanced Topics in Distributed Computing - 3 credits
Prerequisite: MCS5703

A continuation of MCS6723.  Advanced topics in the area of distributed and network 
computing. Topics include routing, addressing, implementations of LANS, and WANS.

MCS6743 Current Developments in Distributed Computing - 3 credits
Prerequisite: MCS6723

Seminar in Distributing Computing, discussion of current papers in the area, 
as well as case studies of current implementations.

MCS7013 Collaborative Research Project 1 - 3 Credits
Prerequisite: Permission of the Program Director

Initiation of work on a large-scale computer science project, involving 
either a student team or a project at the student's workplace.  Students will work 
closely with a faculty member and an industry representative.

MCS7033 Collaborative Research Project 2 - 3 Credits
Prerequisite: MCS7013

Completion of the computer science project begun in Collaborative Research Project 1.
